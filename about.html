Vinyl Vault

Vinyl Vault is a personal, single-file web app for browsing and managing a vinyl record collection synced from Discogs. It's essentially a personal Discogs client focused on browsing and collection-value tracking, built as a single portable HTML file with no backend — all state either comes live from Discogs or lives in the browser's local storage.
More Information Core functionality:
  • Logs in with a Discogs username + personal access token, then pulls your entire collection live from the Discogs API
  • Browsable, filterable grid of album covers — filter by genre, style, format, year, or free-text search; sort by artist, title, year, date added, or estimated value (ascending/descending toggle)
  • A-Z jump navigation for quickly scanning a large collection
  • Clicking any album opens a side panel with tracklist, notes, and label info pulled live from Discogs, instead of leaving the site
Valuation features:
  • Syncs real market price estimates per record from Discogs' marketplace data (requires the user to have a Discogs seller profile)
  • Shows a low/median/high price on each record card
  • Genre-by-genre price breakdown in a dedicated "Price Guide"
  • Running "Estimated Collection Value" total (low/median/high) always visible near the sort controls
  • Once synced, prices are cached locally so it doesn't need to be redone each visit
Extras:
  • "Pick Next Spin" — a randomizer/wheel to help choose what to play next, with spin history tracking
  • "Share & Export" — generates a fully self-contained snapshot HTML file (records + prices baked in) that can be hosted anywhere (e.g. GitHub Pages) so others can browse the collection with no login required
  • Design: dark, warm "record-store" aesthetic (browns/golds, serif display type for the wordmark), responsive down to mobile.
